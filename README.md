### 🧠 Featured Project: superContext

**The Local Brain for your AI Agents.**  
An ultra-fast, local, and agnostic RAG framework designed to give AI coding agents (OpenCode, Cursor, Windsurf, Cline ...) persistent memory and deep semantic understanding of codebases.

*   **Zero Latency & Privacy:** Runs entirely on your machine using **ChromaDB** and **Ollama**.
*   **Semantic Search:** `sc-query` allows agents to find concepts, not just filenames.
*   **Impact Analysis:** `sc-impact` helps agents predict changes before they happen.
*   **Smart Sync:** Native file watcher that updates your index in milliseconds.

🔗 **[Explore superContext →](https://github.com/tu-usuario/superContext)**
